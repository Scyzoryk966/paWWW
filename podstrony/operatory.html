Operatory

W dwóch powyższych linijkach kodu użyliśmy operatora + łączenia stringów. Dzięki niemu możemy wypisywać złożone zdania, zmieniające się w zależności od zmiennych. Jeżeli przy łączeniu różnych typów zmiennych występuje string i operator łączenia +, pozostałe zmienne są przekształcane również na string. Zestawienie operatorów używanych w JavaScript:

Operator Opis Przykłady
Porównania
== jest równe 2==3 wynik: fałsz
!== nie jest równe 2!=3 wynik: prawda
> jest większe 2>3 wynik: fałsz
< jest mniejsze 2<3 wynik: prawda
>= mniejsze lub równe 2>=3 wynik:fałsz
<= większe lub równe 3<=3 wynik: prawda
Przypisania
= x=y
+= x+=7 x=x+7
-= x-=7 x=x-7
*= x*=y x=x+y
/= x/=y x=x/y
%= x%=y x=x%y
Logiczne
&& i true && true = prawda
true $$ false = fałsz
|| lub true || false = prawda
! zaprzeczenie !true = fałsz
