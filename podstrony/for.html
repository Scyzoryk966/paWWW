Pętla For

Stworzenie skryptu który ma po kolei wykonać kilka operacji jest proste, jak to przedstawiłem w poprzednich lekcjach kursu. Jednak nierzadko zdarza się iż pewną operację lub sekwencję operacji warto byłoby wykonać wielokrotnie, chociażby z tego względu aby zaoszczędzić sobie pisania. Owszem można skopiować dany fragment skryptu tyle razy ile potrzeba, ale do jest kiepskie rozwiązanie. Po pierwsze tak przygotowany skrypt jest trudny w utrzymaniu - wyobraź sobie tylko że skopiowałeś(aś) go 20 razy, i musisz w tych 20 miejscach nanieść jakąś poprawkę. Po drugie takie podejście nie zawsze jest możliwe - czasami trzeba powtarzać daną operację, za każdym razem sprawdzając czy określony warunek jest spełniony (np. wczytywać od użytkownika kolejne liczby aż do momentu aż wpisze on liczbę zero). Do rozwiązania tego typu problemów stosuje się pętle. Użycie pętli spowoduje że określony fragment skryptu będzie wykonany określoną z góry ilość razy, lub też do momentu aż podany warunek zostanie spełniony.Wykonamy nastepujacy skrypt:

<script type="text/javascript">
    for (var i=1;i<=50;i++) {
        if(i%2 !=0){
                document.write("<br>");
        }
        else{
            document.write(+i);
        continue
        }
    }
</script>

Program wypisującydaną ilosć kratek:

<script type= "text/JavaScript">
    var e=prompt("Podaj liczbe f " , "" );
    var f;
    for (f=1;f<=e;f++)
    {
        document.write("#");
    }
    document.write("<br>");
</script>

Program wypisujący trójkąt prostokatny o danej podstawie:

<script type="text/javascript">
    var a=prompt("Podaj liczbe a " , "" );
    var c;
    var d;
    var e=1;
    for (c=1;c<=a;c++)
        {
            for (d=1;d<=e;d++)
            {
            document.write("#")
            }
        e++;
        document.write("<br>")
        }
    document.write("<br> <br> <br>");
</script>

Program wypisujący Prostokat o podanych bokach:

<script type= "text/JavaScript">
    var a=prompt("Podaj liczbe a " , "" );
    var b=prompt("Podaj liczbe b " , "" );
    var c;
    var d;
        for (c=1;c<=a;c++)
        {
            for (d=1;d<=b;d++)
            {
                document.write("#")
            }
            document.write("<br>")
        }
    document.write("<br> <br> <br>");
</script>

Program wypisujący tabliczke mnoczenia danych liczb:

<script type= "text/JavaScript">
    var a=prompt("Podaj liczbe a " , "" );
    var b=prompt("Podaj liczbe b " , "" );
    var i;
    var j;
    for(i=a;i<=b;i++) {
        for (j=a;j<=b;j++)
        {
        document.write(j*i+" ")
        }
    document.write("<br>")
    }
</script>
