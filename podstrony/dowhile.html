Pętla do...while

Podobnym rodzajem pętli jest pętla typu do ... while. Zasadniczą różnicą między tym typem a poprzednim jest to, że kod który ma być powtarzany zostanie wykonany przed sprawdzeniem wyrażenia. Ogólna postać tego typu pętli jest następująca:

do {
    ...fragment kodu który będzie powtarzany...
}
while (wyrażenie)

Wykonamy nastepujace zadanie:

 
var x = 1;
var haslo;
do{
haslo = prompt("Podaj hasło...", "...");
if(haslo=="barbarossa"){
document.write("Złamałeś haslo!");
break
}
else{
document.write("Złe hasło!
");
x++
continue
}
}
while(x<4);
if(x==4){
document.write("Przekroczyles ilosc prob zalogowania");
}
